File upload - single
{{ govukTag({ classes: "govuk-!-margin-top-4 govuk-!-margin-bottom-2", text: "MVP" }) }}

This component is intended for use as a minimum viable solution to uploading a single file on a screen.

It allows the user to upload a single file, but has no functionality as part of a series of screens that allows the user to see what they have uploaded, or to remove the uploaded file.

Initial view example 

<div class="govuk-form-group">
  <label class="govuk-label govuk-label--m" for="file-upload-single">
    Drag and drop or choose a file
  </label>
  <input class="govuk-file-upload pins-file-upload" id="file-upload-single" name="file-upload-single" type="file">
</div>

In order for the user the remove or change the uploaded file, the user would have to revisit the page which would change to display a link to the previously uploaded file, and provide the functionality to upload another file to replace it.

When to use this component

The Appeal a planning decision service historically relies on a large amount of documents provided in a number of forms including, but not limited to, Word documents, PDFs, scans of handwritten documents, and photographs.

Where possible, we should look to only include file uploads where it is completely necessary, and where the input of text in the service is not a viable solution.

When not to use this component

When it is in the best interests of the user and the service to collect the relevant information with text inputs.

How it works

This component is an enhancement of the GOV.UK Design System file upload component.

Additional styling has been added to the component to create a recognisable area in which users can drag and drop a file in order to upload it. There are no JavaScript enhancements required for this component.
