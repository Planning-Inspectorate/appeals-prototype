File upload - single
{{ govukTag({ classes: "govuk-!-margin-top-4 govuk-!-margin-bottom-2", text: "MVP" }) }}

This component is intended for use as a minimum viable solution to uploading a single file on a screen.

It allows the user to upload a single file, but has no functionality as part of a series of screens that allows the user to see what they have uploaded, or to remove the uploaded file.

In order for the user the remove or change the uploaded file, they would have to revisit the page which would change to display a link to the previously uploaded file, and provide the functionality to upload another file to replace it.

Example 


<div class="govuk-form-group">
  <label class="govuk-label govuk-label--m" for="file-upload-single">
    Drag and drop or choose a file
  </label>
  <input class="govuk-file-upload pins-file-upload" id="file-upload-single" name="file-upload-single" type="file">
</div>
