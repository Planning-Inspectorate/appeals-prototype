Find and comment on an appeal

This is a service for searching, viewing and commenting on planning appeals. Commenting on a planning appeal is also known as ‘making a representation’.

You can use this service to:

  • view appeals that are open for comments and submit your comments
  • view appeals that are closed for comments
  • view appeals that are complete and view the decision for those appeals

If you want to comment on an appeal, you'll need an email address so we can contact you if we have any questions.

You will also need (one of the following):

  • appeal site postcode
  • appeal site grid reference
  • 7-digital appeal reference

Guidance about your comments

Any previous comments that you submitted as part of an application will be considered by the Planning Inspector. The local planning authority shares these with the Planning Inspectorate, so there is no need to submit repeat comments.

You must not include sensitive information. If you do include sensitive information, we’ll send them back to you. Comments that include sensitive information will not be included in the appeal.
